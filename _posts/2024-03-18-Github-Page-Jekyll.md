## 安裝
- 安裝ruby on windows 後執行以下指令
``` js
gem install jekyll bundler
```

## 執行

- 前往專案
``` js
cd (github.io repo path)
```

- 本地建立
``` js
bundle exec jekyll serve
```

- prod環境建立
``` js
bundle exec jekyll build
```
